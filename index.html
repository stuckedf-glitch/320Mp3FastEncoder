// Documentation
Operating Manual

Interface & Navigation

  • Drag & Drop — Drop files onto the central area (dotted rectangle). Accepts multiple files simultaneously.
  • Manual Selection — Click the center area to open the system file picker.
  • ▶ Play — Starts the conversion queue.
  • ■ Stop — Immediately interrupts the current process and clears the queue.
  • ✎ Tags — Opens the ID3v1 metadata editor for batch application.

Tag Editing — ID3v1 Editor

Clicking the edit icon opens a side panel where you can set: Title, Artist, Album, Year, Genre, and Comment.

Important: All fields defined in the editor will be applied to all files in the current queue after clicking APPLY.

Conflict Management

When an output file already exists in the destination folder, a dialog presents three options:

  1. Overwrite — Replaces the existing file.
  2. Keep Both — Renames the new file (e.g. song_1.mp3).
  3. Skip — Skips conversion for that specific file.

Workflow Algorithm

  1. Ingestion — Paths received via Drag & Drop are normalized by the app.
  2. Metadata Mapping — If tags are set, -metadata flags are generated for the FFmpeg command.
  3. Duration Analysis — FFmpeg pre-reads each file to obtain exact duration in seconds for the progress bar.
  4. Transcoding — Input decoded in real time; psychoacoustic modeling removes inaudible frequencies; data written to disk at 320kbps CBR.
